    \item Can you clarify the steps to prove a proper subset?
    🔗
    Answer: Yes! To shows \(A\subset B\text{,}\) show that \(A\subseteq B\) and \(A\neq B\text{.}\)
    🔗
    1. To show that \(A\subseteq B\text{,}\) let \(x\in A\) (choosing an arbitrary element of \(A\text{,}\) so use the rule of \(A\) and variables to define) and show that \(x\in B\) (show that the arbitrary \(x\) follows the rule for \(B\)).
      🔗
      🔗
    2. To show \(A\neq B\text{,}\) show there is some element \(B\) that is not in \(A\text{.}\) (You could think of this as \(B\) is ``strictly bigger" than \(A\text{.}\)) Pick a specific element for this part, e.g., like \(5\) is in \(B\) but \(5\notin A\text{.}\) Make sure to show why it satisfies the rule for \(A\) but not the rule for \(B\text{.}\)
      🔗
      🔗
    If you’re trying to do a proper subset proof, it’s a good idea to list the elements in roster notation first. This will help you find an element that shows \(A\neq B\text{.}\) Note listing the elements in roster notation is not a proof!
    🔗
    Practice Question: Prove that if \(A = \{ x\in\mathbb{Z} : x = 12k+1 \text{ for some } k\in\Z\} \) and \(B =\{x\in \Z : x\equiv 1\mod {4}\}\) then \(A\subset B\text{.}\)

    \item Is there a trick to work out how to get from \(S_k\) to \(S_{k+1}\) when the statement isn’t a infinite sum? (for example, if the statement is based on divides such as \(2\) divides \(n^2 - n\)).
    🔗
    Answer: Induction can be used in a lot of contexts, so it’s not necessarily possible to say a technique that will work every time. The sums are a common place to start with induction. The next place would be things that have associated equations - like \(2 \mid n^2 - n\) turns into the equation \(2j = n^2-n\) for some integer \(j\text{.}\) When you are using equations, consider your \(S_k\) and \(S_{k+1}\text{.}\) In this case \(S_k\) is \(2j = k^2-k\) for some integer \(k\text{,}\) and \(S_{k+1}\) is \(2j' = (k+1)^2 -(k+1)\text{.}\) In this proof, like in even/odd proofs, you can start with one side of \(S_{k+1}\text{.}\) E.g., start with \((k+1)^2 -(k+1)\text{.}\) Expand, and then look for \(k^2-k\) somewhere in your expression!
    🔗
    Practice Question: Prove using induction that for all \(n\in\mathbb{N}\text{,}\) \(6 \mid (5^{2n}-1)\text{.}\) (See footnote for a hint.)
     7 
    After simplifying one side of \(S_{k+1}\) you may have \(5^{2k+2} -1\text{.}\) You want to look for \(5^{2k}-1\) in this expression. Use exponent rules to split up the \(5^{2k+2}\) into \(5^j5^{2k} \) for some \(j\) Keep reading for more: \(25\cdot 5^{2k}=24\cdot 5^{2k}+5^{2k}\text{.}\)
    🔗
    \item How does assuming \(S_1, S_2, S_3, ..., S_n\) using strong induction instead of just \(S_n\) help prove things?
    🔗
    Answer: We’ll do strong induction examples on pages 54 and 55. As one example you’ve already seen, in the Fundamental Theorem of Arithmetic, say you are doing the induction step where \(k=54\text{.}\) So you want to show \(55\) is either prime or a product of primes. It’s not enough to just know that \(54\) is prime or a product of primes. The proof says - ok \(k+1=55\) is not prime, so we can write \(55 = a\cdot b\) where \(1<a,b<55\text{.}\) In this case \(a=11\) and \(b=5\text{.}\) So you need to know that both \(11\) and \(5\) are either prime or a product of primes. That’s why it helps to have assumed \(S_2, S_3, \dots, S_5, \dots, S_{11},\dots, S_{54}\text{,}\) because \(5\) and \(11\) are in that list.
    🔗
    Practice Question: Prove that for all \(n\in\mathbb{N}\text{,}\) \(f_n < 2^n\) where \(f_n\) is the \(n^{th}\) Fibonacci number.
    🔗
    \item How do you do proofs with Cartesian products?
    🔗
    Answer: If you are trying to prove a Cartesian Product of two sets is a subset or equal to Cartesian Product of another two sets, you still use the choose an element method. In this case, when you choose an element of one set, that element is an ordered pair. Here’s an example.
    🔗

    Theorem 30.

    Let \(A,B,\) and \(C\) be subsets of some universal set \(U\text{.}\) If \(A\subseteq B\) then \(A\times B\subseteq A\times C\text{.}\)
    🔗
    🔗

    Proof.
    Suppose \(A,B,\) and \(C\) be subsets of some universal set \(U\) and \(A\subseteq B\text{.}\) We will show \(A\times B\subseteq A\times C\text{.}\) Let \(x\in A\times B\text{.}\) Then \(x = (a,c)\) for some \(a\in A\) and \(c\in C\text{.}\) Since \(a\in A\) and \(A\subseteq B\text{,}\) by the definition of subset, we know \(a\in B\text{.}\) Thus \(x=(a,c) \in B\times C\) as \(a\in B\) and \(c\in C\text{.}\) Thus, we’ve shown that if \(x\in A\times C\) then \(x\in B\times C\text{.}\) By the definition of subset, \(A\subseteq B\) then \(A\times B\subseteq A\times C\text{.}\)
    🔗
    🔗
    Practice Question: Is \((A\times B) \cup (C\times D) = (A\cup C) \times (B\cup D)\text{?}\) Prove or disprove.
    🔗
    \item Can we do another example of a proof with sets (union, intersection, etc.) where the elements of \(A\) and \(B\) are not given (arbitrary sets)?
    🔗
    Answer: Sure! Suppose \(U\) is some universal set and \(A,B,\) and \(C\) are subsets of that universal set. Suppose \(A\subseteq C\) and \(B\subseteq C\text{.}\) Show \(A\cup B \subseteq C\text{.}\)
    🔗

    Theorem 31.

    Let \(U\) be a universal set and \(A, B,\) and \(C\) be subsets of \(U\text{.}\) If \(A\subseteq C\) and \(B\subseteq C\) then \(A\cup B \subseteq C\text{.}\)
    🔗
    🔗

    Proof.
    To prove this, we’ll use the ``choose an element method". Let \(x\in U\) such that \(x\in A\cup B\text{.}\) We will show \(x\in C\text{.}\) Since \(x\in A \cup B\text{,}\) we know \(x\in A\) or \(x\in B\text{.}\) So we’ll consider two cases: (1) \(x\in A\) and (2) \(x\in B\text{.}\)
    🔗
    Case 1: Suppose \(x\in A\text{.}\) Then, by the definition of subset, because \(A\subseteq C\) and \(x\in A\text{,}\) we know \(x\in C\text{,}\) which is what we wanted to show.
    🔗
    Case 2: Suppose \(x\in B\text{.}\) Then, by the definition of subset, because \(B\subseteq C\) and \(x\in B\text{,}\) we know \(x\in C\text{,}\) which is what we wanted to show.
    🔗
    In both cases, \(x\in C\text{.}\) So, assuming \(A\subseteq B\text{,}\) \(B\subseteq C\text{,}\) if \(x\in A\cup B\) then \(x\in C\text{.}\) Thus, by the definition of subset \(A\cup B \subseteq C\text{.}\)
    🔗
    🔗
    Practice Question: Let \(A,B\) and \(C\) be subsets of some universal set \(U\text{.}\) Show that \(A\cap (B\cup C) = (A\cap B) \cup (A\cap C)\text{.}\)
    🔗
    \end{enumerate}
