    Subsection What is a Set?

    A set is an unordered collection of distinct objects which are called elements.
    🔗
    🔗

    Subsection The Symbols \(\in, \notin, \subseteq, \subset, \not\subseteq, = \)

    • If \(x\) is an element of a set \(S\text{,}\) we write \(x\in S\text{.}\) This is read ``\(x\) in \(S\)".
      🔗
      🔗
    • If \(x\) is not an element of a set \(S\text{,}\) we write \(x\notin S\text{.}\) This is read as ``\(x\) is not in \(S\)".
      🔗
      🔗
    • If every element of \(A\) is also an element of \(B\) then \(A\) is a subset of \(B\text{,}\) denoted \(A\subseteq B\text{.}\)
      🔗
      🔗
    • If \(A\subseteq B\) and \(A\neq B\text{,}\) then \(A \subset B\text{,}\) we say ``\(A\) is a proper subset of \(B\)".
      🔗
      🔗
    • If \(A\) is not a subset of \(B\) then we write \(A\not\subseteq B\text{.}\) This means there exists an \(x\in A\) such that \(x\notin B\text{.}\)
      🔗
      🔗
    • If \(A\) and \(B\) have the exact same elements, we write \(A=B\text{.}\) This is read as ``\(A\) equals \(B\)".
      🔗
      🔗

    Subsection Roster Notation and Set Builder Notation

    • Roster notation is when we list all the elements, or list enough elements so that pattern is obvious and use \(\dots\text{.}\) E.g., \(\{1,4,9,16,25,\dots\}\text{.}\)
      🔗
      🔗
    • Set builder notation is when we define the elements by a rule. E.g., \(\{n^2: n\in\mathbb{N}\}\text{.}\)
      🔗
      🔗

    Subsection Proving \(A\subseteq B\) - The ``Choose an Element Method"

    The notation \(A\subseteq B\) is read ``\(A\) is a subset of \(B\)" and means that if \(x\in A\) then \(x\in B\text{,}\) that is, every element of \(A\) is also an element of \(B\text{.}\)
    🔗
    Thus, if we want to prove that \(A\subseteq B\text{,}\) we are proving a conditional statement, i.e., an ``if - then" statement. We’ll do a direct proof of this conditional statement. So, we’ll assume the hypothesis, and show the conclusion.
    🔗
    More specifically, we’ll start with an arbitrary \(x\) such that \(x\in A\text{,}\) and we’ll show \(x\in B\text{,}\)
    🔗

    Activity 15.

    Prove the following theorem.
    🔗
    🔗

    Theorem 18.

    If \(A = \{x\in\mathbb{Z} \mid x \equiv 2 \pmod{4}\} \) and \(B =\{x\in\mathbb{Z} \mid x \equiv 0 \pmod{2}\}\) then \(A\subseteq B\text{.}\)
    🔗
    🔗
    Start with a pre-proof analysis. Consider listing the elements in \(A\) and the elements in \(B\) to see if you believe the conjecture. Then, choose an arbitrary element of \(A\text{,}\) e.g., ``Let \(x\in A\text{.}\) So \(x\in \mathbb{Z}\) and \(x\equiv 2 \pmod{4}\text{.}\)" Work to show that this \(x\) is in \(B\text{,}\) that is \(x\equiv 0 \pmod {2}\text{.}\)
    🔗
    (If time:) It’s actually true that \(A\subset B\text{.}\) What would you add to your proof to prove this?
    🔗
    🔗

    Subsection Proving \(A = B\)

    To say \(A = B\) is to say that \(A\) and \(B\) contain exactly the same elements. Said differently, it means these two things:
    🔗
    1. Every element in \(A\) is also in \(B\) (which means \(A\subseteq B\)), and
      🔗
      🔗
    2. Every element in \(B\) is also in \(A\) (which means \(B\subseteq A\)).
      🔗
      🔗
    Indeed, a slick way to prove that \(A=B\) is to prove both \(A\subseteq B\) and \(B\subseteq A\text{,}\) both of which can be done using the ``choose an element" method.
    🔗
